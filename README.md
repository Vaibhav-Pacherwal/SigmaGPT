# SigmaGPT

A full-stack AI chat platform built with React, Node.js, MongoDB, and Groq API.

SigmaGPT provides real-time conversational AI with persistent chat threads, a modern interface, and scalable backend architecture.

## Live Demo

Frontend: [Live App](https://sigma-gpt-theta.vercel.app)  
Backend: [API](https://sigmagpt-37ow.onrender.com)

---

## Features

- AI-powered conversations using Groq API
- Thread-based chat management
- Persistent conversation history
- Modern responsive UI
- MongoDB integration
- Fast backend API handling
- Real-time prompt-response flow

## Installation

### Clone Repository

```bash id="nyx5q5"
git clone YOUR_REPO_URL
cd SigmaGPT
```

### Backend Setup

```bash id="bn4yeh"
cd backend
npm install
npm start
```

### Frontend Setup

```bash id="n9h6qi"
cd frontend
npm install
npm run dev
```
